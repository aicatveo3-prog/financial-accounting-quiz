### 🔍 이 문제가 묻는 핵심
기말 재고자산 실사 금액(₩100,000)에 **추가로 포함해야 할 항목**을 판단하여 정확한 기말재고를 구하고, 그로부터 매출원가를 계산하는 문제이다.

### 📖 지문 하나하나 뜯어보기
**"미착상품 ₩60,000 (도착지 인도조건으로 매입, 운송 중)"**
도착지 인도조건(FOB Destination)으로 **매입**한 것이다. 도착지 인도조건에서는 물건이 **도착해야** 소유권이 넘어온다. 아직 운송 중이므로 소유권이 아직 **판매자**에게 있다. → **기말재고에 포함하지 않는다.**
**"적송품 ₩200,000 (80% 판매 완료)"**
적송품은 위탁판매를 위해 수탁자에게 보낸 상품이다. 80%가 판매 완료되었으므로, **판매되지 않은 20%**만 우리의 기말재고이다.
```
미판매 적송품 = ₩200,000 × 20% = ₩40,000 → 기말재고에 포함
```
**"시송품 ₩60,000 (매입의사표시 금액 ₩20,000)"**
시용판매를 위해 고객에게 보낸 상품 중, ₩20,000은 고객이 사겠다고 했으므로 판매 성립(기말재고 제외). 나머지 ₩40,000은 매입의사를 표시하지 않았으므로 아직 우리 것이다.
```
매입의사 미표시 시송품 = ₩60,000 − ₩20,000 = ₩40,000 → 기말재고에 포함
```

### 📐 풀이 과정
**STEP 1 : 정확한 기말재고**
```
정확한 기말재고 = ₩100,000(실사 금액)
               + ₩40,000(미판매 적송품)
               + ₩40,000(매입의사 미표시 시송품)
              = ₩180,000
```
미착상품(₩60,000)은 도착지 인도조건이므로 포함하지 않는다!
**STEP 2 : 매출원가**
```
매출원가 = ₩200,000(기초재고) + ₩1,000,000(당기매입) − ₩180,000(기말재고)
        = ₩1,020,000
```
**정답 : ④ ₩1,020,000**

### 💡 함께 알아두면 좋은 것
**"부분적으로 판매된 경우"를 처리하는 법**
적송품이나 시송품이 **일부만 판매**되었을 때, **판매된 부분은 기말재고에서 제외**하고 **미판매 부분만 포함**한다. 이 문제처럼 비율이나 금액이 주어지면, 미판매분만 정확히 계산해야 한다.
**인도조건별 매입/판매 시 기말재고 포함 판단 요약**
| 상황 | 선적지 인도 | 도착지 인도 |
|:---|:---:|:---:|
| **매입**한 상품 운송 중 | ✅ 포함 (소유권 넘어옴) | ❌ 제외 (아직 안 넘어옴) |
| **판매**한 상품 운송 중 | ❌ 제외 (소유권 넘어감) | ✅ 포함 (아직 안 넘어감) |
