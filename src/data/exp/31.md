### 🔍 이 문제가 묻는 핵심
기말 재고자산 실사액에 **포함되지 않은 항목**들을 검토하여, 올바른 기말재고자산 금액을 구하는 문제이다. "기말재고에 포함해야 하는데 빠진 것은 무엇인가?"를 판단하는 것이 핵심이다.

### 📖 기말재고 포함 여부 판단 기준
기말재고에 포함할지 말지의 기준은 **"결산일 현재 소유권이 누구에게 있는가?"**이다. 물건이 물리적으로 내 창고에 있느냐가 아니라, **법적 소유권**이 나에게 있느냐가 기준이다.

### 📖 지문 하나하나 뜯어보기
**① "FOB 선적지 인도조건으로 구입한 상품 ₩15,000이 운송 중"**
FOB 선적지 인도조건(FOB Shipping Point)이란, 판매자가 물건을 **선적(배에 실은) 시점**에 소유권이 매입자에게 넘어가는 조건이다. 이미 선적되었으므로 소유권은 **우리 회사(매입자)**에게 있다. 물건이 아직 도착하지 않았더라도 **기말재고에 포함해야 한다.**
→ **+ ₩15,000 가산**
**② "FOB 도착지 인도조건으로 판매한 상품 ₩20,000이 운송 중"**
FOB 도착지 인도조건(FOB Destination)이란, 물건이 **도착한 시점**에 소유권이 매입자에게 넘어가는 조건이다. 아직 도착하지 않았으므로 소유권은 여전히 **우리 회사(판매자)**에게 있다. 이미 팔았다고 생각했지만, 아직 소유권이 넘어가지 않았으므로 **기말재고에 포함해야 한다.**
→ **+ ₩20,000 가산**
**③ "수탁자가 판매하지 못한 위탁품 ₩30,000"**
위탁판매란, 다른 회사(수탁자)에게 물건을 맡겨서 대신 팔아달라고 하는 것이다. 수탁자가 아직 팔지 못했다면, 그 물건의 소유권은 여전히 **우리 회사(위탁자)**에게 있다. 물건이 수탁자의 창고에 있더라도 **우리의 기말재고에 포함해야 한다.**
→ **+ ₩30,000 가산**
**④ "고객이 매입의사를 표시하지 않은 시송품 ₩20,000"**
시용판매(시송품)란, 고객에게 물건을 보내서 써보게 하고, 마음에 들면 사겠다고 하는 판매 방식이다. 고객이 **매입의사를 표시하지 않았다면**, 아직 판매가 성립되지 않은 것이므로 소유권은 여전히 **우리 회사**에게 있다. **기말재고에 포함해야 한다.**
→ **+ ₩20,000 가산**

### 📐 풀이 과정
```
올바른 재고자산 = ₩100,000(실사재고액)
               + ₩15,000(선적지 인도조건 매입 → 소유권 이전됨)
               + ₩20,000(도착지 인도조건 판매 → 소유권 아직 우리)
               + ₩30,000(미판매 위탁품 → 소유권 우리)
               + ₩20,000(매입의사 미표시 시송품 → 소유권 우리)
              = ₩185,000
```
**정답 : ④ ₩185,000**

| **미착상품 (도착지 인도)** | ❌ 제외 | 도착해야 소유권 이전 → 아직 상대방 것 |

| **선적지 인도로 판매, 운송 중** | ❌ 제외 | 선적 시 소유권 넘어감 → 이미 상대방 것 |
| **미판매 위탁품(적송품)** | ✅ 포함 | 수탁자가 안 팔았으면 소유권 내 것 |
| **판매된 위탁품** | ❌ 제외 | 수탁자가 팔았으면 소유권 넘어감 |

| **매입의사 표시한 시송품** | ❌ 제외 | 고객이 사겠다고 하면 판매 성립 |
| **담보제공 재고** | ✅ 포함 | 담보 제공해도 소유권은 여전히 내 것 |
