### 🔍 이 문제가 묻는 핵심
감모손실에서 **단위당 원가를 역산**하고, 평가손실에서 **순실현가능가치를 역산**하는 2단계 역산 문제이다.

### 📐 풀이 과정
**STEP 1 : 단위당 원가 역산**
```
₩50,000 = (1,100개 − 1,000개) × 단위당 원가
₩50,000 = 100개 × 단위당 원가
단위당 원가 = ₩500
```
**STEP 2 : 순실현가능가치 역산**
```
₩40,000 = 1,000개 × (₩500 − 순실현가능가치)
₩500 − 순실현가능가치 = ₩40
순실현가능가치 = ₩460
```
**정답 : ② ₩460**

### 💡 함께 알아두면 좋은 것
**이 유형의 풀이 순서 : 항상 감모 먼저, 평가 나중**
감모손실에서 단위당 원가를 알아낸 뒤, 그 원가를 평가손실 공식에 넣어 순실현가능가치를 구한다. 순서가 반대면 풀 수 없다.
```
감모손실 → 단위당 원가 도출 → 평가손실에 대입 → 순실현가능가치 도출
```
