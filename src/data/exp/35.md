### 🔍 이 문제가 묻는 핵심
**재고자산감모손실**의 개념을 이해하고, 감모손실 금액으로부터 **실지재고 수량**을 역산하는 문제이다.

### 📖 핵심 개념 : 재고자산감모손실이란?
재고자산감모손실이란, 장부에 기록된 수량보다 **실제로 세어본 수량이 적을 때** 발생하는 손실이다. 쉽게 말해, 물건이 도난·파손·증발 등으로 **줄어든 것**이다.
```
감모손실 = (장부수량 − 실사수량) × 단위당 원가
```

### 📐 풀이 과정
```
장부수량 = ₩200,000 ÷ ₩1,000 = 200개
₩30,000 = (200개 − 실사수량) × ₩1,000
200개 − 실사수량 = 30개
실사수량 = 170개
```
**정답 : ③ 170개**

### 💡 함께 알아두면 좋은 것
**감모손실 vs 평가손실 — 이 둘은 완전히 다르다!**
| | 감모손실 | 평가손실 |
|:---|:---|:---|
| **원인** | 물건의 **수량**이 줄어듦 (도난, 파손 등) | 물건의 **가치**가 떨어짐 (시가 하락) |
| **공식** | (장부수량 − 실사수량) × 단위원가 | 실사수량 × (단위원가 − 순실현가능가치) |
| **수량 변화** | 있음 (줄어듦) | 없음 (수량은 그대로) |
| **단가 변화** | 없음 (원가 그대로) | 있음 (시가가 원가보다 낮음) |
