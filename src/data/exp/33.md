### 🔍 이 문제가 묻는 핵심
재고자산 실사 금액(₩110,000)에 **반영되지 않은 사항들**을 고려하여 정확한 기말재고를 구하고, 매출원가를 계산하는 문제이다.

### 📖 지문 하나하나 뜯어보기
**① "도착지 인도조건으로 매입한 상품 ₩20,000이 운송 중"**
도착지 인도조건에서는 물건이 **도착해야** 소유권이 넘어온다. 아직 운송 중이므로 소유권은 판매자에게 있다. → **기말재고에 포함하지 않는다.** (이미 실사에도 포함되지 않았으므로 조정 불필요)
**② "시용판매 상품 ₩40,000 중 50%에 대해 구매의사 표시"**
- 구매의사를 표시한 50% (₩20,000) → 판매 성립 → 기말재고 제외
- 구매의사를 표시하지 않은 50% (₩20,000) → 아직 우리 것 → **기말재고에 포함**
시송품은 고객에게 보냈으므로 실사에서 빠져있다. 따라서 매입의사 미표시분(₩20,000)을 기말재고에 **가산**해야 한다.
**③ "담보로 제공한 상품 ₩50,000, 실사 시 포함됨"**
담보로 제공한 재고자산은 소유권이 **여전히 우리 회사**에게 있다. 담보는 빌려준 것이 아니라, 채무의 보증으로 제공한 것이므로 소유권은 변하지 않는다. 이미 실사에 포함되어 있으므로 **추가 조정 불필요**. 올바르게 처리된 상태이다.

### 📐 풀이 과정
**STEP 1 : 정확한 기말재고**
```
정확한 기말재고 = ₩110,000(실사 금액)
               + ₩20,000(매입의사 미표시 시송품)
              = ₩130,000
```
**STEP 2 : 매출원가**
```
매출원가 = ₩100,000(기초재고) + ₩200,000(당기매입) − ₩130,000(기말재고)
        = ₩170,000
```
**정답 : ① ₩170,000**

### 💡 함께 알아두면 좋은 것
**담보제공 재고자산은 소유권 변동이 없다!**
담보로 제공했다고 해서 소유권이 넘어가는 것이 아니다. 담보는 "빚을 못 갚으면 이걸로 대신하겠다"는 보증일 뿐이다. 따라서 담보제공 재고자산은 **우리의 기말재고에 포함**해야 한다. 이 문제에서는 이미 실사에 포함되어 있었으므로 별도 조정이 필요 없었다.
