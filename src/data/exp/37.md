### 🔍 이 문제가 묻는 핵심
감모손실과 평가손실이 **주어져 있을 때**, 거꾸로 **실사수량**과 **순실현가능가치**를 **역산**하는 문제이다.

### 📐 풀이 과정
**STEP 1 : 장부수량**
```
장부수량 = ₩200,000(장부상 기말재고) ÷ ₩2,000(단위당 원가)
        = 100개
```
**STEP 2 : 실사수량 역산 (감모손실에서)**
```
₩20,000 = (100개 − 실사수량) × ₩2,000
100개 − 실사수량 = 10개
실사수량 = 90개
```
**STEP 3 : 순실현가능가치 역산 (평가손실에서)**
```
₩18,000 = 90개 × (₩2,000 − 순실현가능가치)
₩2,000 − 순실현가능가치 = ₩200
순실현가능가치 = ₩1,800
```
**정답 : ③ 실사수량 90개 / 순실현가능가치 ₩1,800**

### 💡 함께 알아두면 좋은 것
**검증 : 3단계 재고 흐름**
```
장부재고 = 100개 × ₩2,000 = ₩200,000
실사재고 = 90개 × ₩2,000  = ₩180,000  ← 감모손실 ₩20,000 ✔
저가재고 = 90개 × ₩1,800  = ₩162,000  ← 평가손실 ₩18,000 ✔
```
