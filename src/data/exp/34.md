### 🔍 이 문제가 묻는 핵심
기말재고 실사에서 발생한 **오류들이 매출원가에 미치는 영향**을 분석하는 문제이다. 실지재고조사법에서 매출원가 = 기초재고 + 당기매입 − **기말재고**이므로, 기말재고가 과소계상되면 매출원가는 과대계상되고, 그 반대도 마찬가지이다.

### 📐 풀이 과정
| 항목 | 기말재고 영향 | 매출원가 영향 |
|:---|:---:|:---:|
| ① 시송품 (매입의사 있는데 포함됨) | **₩1,000,000 과대** | ₩1,000,000 과소 |
| ② 선적지 매입 (실사 누락) | **₩1,100,000 과소** | ₩1,100,000 과대 |
| ③ 도착지 매입 (도착했는데 누락) | **₩1,000,000 과소** | ₩1,000,000 과대 |
```
기말재고 순 영향 = −₩1,000,000(과대) + ₩1,100,000(과소) + ₩1,000,000(과소)
               = ₩1,100,000 과소계상
매출원가 순 영향 = ₩1,100,000 과대계상
```
기말재고가 과소 → 매출원가는 과대 (기말재고와 매출원가는 반대 방향!)
**정답 : ① ₩1,100,000 과대계상**

### 💡 함께 알아두면 좋은 것
**기말재고 오류와 매출원가의 관계**
```
매출원가 = 기초재고 + 당기매입 − 기말재고
```
기말재고가 과소계상되면 → 빼는 값이 줄어들어 → 매출원가 **과대**계상
기말재고가 과대계상되면 → 빼는 값이 늘어나 → 매출원가 **과소**계상
기말재고와 매출원가는 **반대 방향**으로 움직인다!
